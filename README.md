# StarButterfly

Add a little Mewnian magic to your terminal.

## Installation

    $ gem install starbutterfly

## Usage

	$ starbutterfly cast
	*Cast a Mewnian spell right into your terminal*

	$ starbutterfly tweet
	*Cast a Mewnian spell through a portal to the Twitterverse*
